Browser JavaScript Test

Instructions:

  1. Open this page in a WordPress page with the [userinfo_check] shortcode
  2. Open browser console (F12)
  3. Run the following commands in the console:

Test Commands:

// 1. Check if jQuery is loaded
typeof jQuery
// Should return: "function"

// 2. Check if UserinfoManager is loaded
typeof UserinfoManager
// Should return: "object"

// 3. Check if userinfo_l10n is defined
typeof userinfo_l10n
// Should return: "object"

// 4. Check ajax_url
userinfo_l10n.ajax_url
// Should return: "http://localhost/formwp/wp-admin/admin-ajax.php" or similar

// 5. Check if form exists
$('#userinfo-check-form').length
// Should return: 1

// 6. Check if form has submit handler
$._data($('#userinfo-check-form')[0], 'events')
// Should show 'submit' event

// 7. Manual test submission
$('#userinfo-check-form').trigger('submit')
// Should trigger AJAX or console log

// 8. Check if JavaScript file loaded
console.log('JavaScript file check - look in Network tab for userinfo-frontend.js')

What to Look For:

  • If jQuery is "undefined": jQuery not loading (WordPress issue)
  • If UserinfoManager is "undefined": JavaScript file not loading
  • If userinfo_l10n is "undefined": Localization not working
  • If form length is 0: Form not rendered
  • If no submit event: Handler not attached
